# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/),
and this project adheres to [Semantic Versioning](https://semver.org/).

## [0.1.0] - 2026-03-26

### Added

- Initial release — migrated from standalone `mcporter-bridge-lite`
- `mcporter_list_servers` — list MCP servers known to mcporter
- `mcporter_help` — show tool schema/docs for one MCP server
- `mcporter_call_tool` — call one mcporter-backed MCP tool
- Converted to TypeScript, integrated into `@codon` monorepo
